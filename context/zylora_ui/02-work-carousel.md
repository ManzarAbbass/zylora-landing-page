# Section: Our Work Speaks For Itself (Carousel)

## Layout
- Full-bleed purple rounded section, large repeating background wordmark pattern (faint, decorative).
- Centered heading: "Our Work Speaks *For Itself*" (white serif, italic accent word in yellow).
- Horizontal carousel of phone-mockup screenshots (5 visible: 2 blurred/faded on each side for depth, 1 sharp center card).
- Center card: full email/landing-page mockup screenshot inside phone frame, diagonal ribbon/tag overlay showing a stat callout (e.g. discount code + a fabricated placeholder metric string).
- Carousel dot indicators below (active dot highlighted yellow, elongated pill shape vs round dots).
- CTA button below carousel, yellow fill, black text, arrow icon — different accent color (yellow) from rest of page (purple), used to stand out against purple background.

## Content structure
- Section heading only, no body copy — carousel + mockups carry the content.
- Each carousel slide = one client/portfolio piece screenshot (email design work samples).

## Design notes
- Carousel center-focus pattern: adjacent slides shown blurred/scaled down to indicate more content, common "featured work" pattern.
- Ribbon/tag overlay on center mockup is a design flourish highlighting a metric — should use only real, verifiable numbers if reused.
- Yellow CTA breaks from purple/violet palette used elsewhere — used specifically for contrast against solid purple background.

## Reference-only note
One of the placeholder metric strings visible in this screenshot is clearly broken/nonsensical filler text, not a real number — a reminder that template mockups often contain placeholder data that must never be copied as-is into real content.
