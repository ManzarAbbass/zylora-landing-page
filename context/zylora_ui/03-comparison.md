# Section: Comparison — "The Truth About Your Numbers"

## Layout
- White background, pill badge centered above heading ("THE TRUTH ABOUT YOUR NUMBERS").
- Centered serif heading, 2 lines, one phrase in purple italic (dashboard/tool name).
- Subtext paragraph centered below heading, 2 lines, muted gray.
- Two-column comparison card layout below:
  - Left card: light gray bg, header "THE TYPICAL AGENCY OR FREELANCER" (negative framing), 4 stacked sub-cards each with icon (crossed-out/negative icon in purple circle), bold title, gray description.
  - Right card: solid purple bg, header "[BRAND] MARKETING" (positive framing), 4 stacked sub-cards each with icon (yellow circle), white bold title, light purple/white description.
- CTA button centered below both cards, purple fill, white text, arrow icon.

## Content structure
- Pattern: problem/pain point (left) directly paired with the brand's counter-claim (right), same 4-row structure both sides for easy visual comparison.
- Left card pain points: vanity metrics, one-size-fits-all flows, poor design work, AI-generated copy.
- Right card counter-claims: proprietary data access, purchase-timing AI tracking, direct client collaboration, human-designed/human-written work.

## Design notes
- Icon color coding: purple circle icons = negative/left column, yellow circle icons = positive/right column — reinforces which side is "good."
- This "us vs. them" comparison format is a strong persuasion pattern — direct antithesis pairing, one-for-one rebuttal structure.

## Reference-only note
This entire section's rhetorical structure (competitor-shaming framed as "the typical agency/freelancer") depends on unverifiable claims about competitors and about the brand's own capabilities (e.g. "proprietary data nobody else has"). If adapting this pattern for a real project, every right-column claim needs to be something actually true and demonstrable — this format is persuasive precisely because it sounds authoritative, which makes it easy to overclaim.
