# Section: Educational Videos (YouTube Content Carousel)

## Layout
- White background, pill badge left ("EMAIL MARKETING INSIGHTS").
- Serif heading left-aligned: "Over 35 *Educational* Videos" (italic purple emphasis), subtext below ("Checkout the latest in the E-commerce Email Marketing space").
- Purple CTA button top-right, same repeated style.
- 3-column card grid: YouTube-thumbnail-style images (bold clickbait-style text overlay on each thumbnail, e.g. big colored headline text, presenter face reacting to camera).
- Below each thumbnail: colored bar/banner with promo text (giveaway/incentive callout), attribution line (creator/channel name) beneath.

## Content structure
- Each card: thumbnail (video-platform style, high-contrast bold text hooks) + repeated promotional banner text + channel/creator attribution.
- All 3 visible cards share identical promo banner text — a giveaway CTA — suggesting this banner is a persistent overlay across all videos rather than per-video content.

## Design notes
- Thumbnail style deliberately mimics native YouTube thumbnail conventions (bold sans headline type, high contrast, presenter face with visible reaction) — different visual language from rest of the site's clean editorial style, because it's meant to be recognized as "YouTube content" at a glance.
- Badge label reuse (same pill text as section 4) looks like a copy/paste inconsistency in the source template rather than intentional.

## Reference-only note
Layout pattern (thumbnail grid + carousel nav) is reusable; the mismatched badge label is a small but real inconsistency worth avoiding — labels should describe the section they're actually attached to.
