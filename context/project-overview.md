# Zylora — Project Overview

> A fictional email marketing agency landing page
> built as a portfolio project to demonstrate
> landing page development skills.
> Stack: Next.js 15 · TypeScript · Tailwind CSS · GSAP + ScrollTrigger · Embla Carousel · Vercel

---

## Purpose

Demonstrate ability to build a modern, animated
marketing agency landing page with complex UI
components including carousels, accordions,
and booking integrations.

> ⚠️ This is a fictional brand — all content,
> clients, and testimonials are fabricated for
> demo purposes only.

## Rules

> ⚠️ Never use Myria branding, logo, or content
> ⚠️ All clients and testimonials must be fictional
> ⚠️ Never add sections not listed above
> ⚠️ All files must be TypeScript — no .js files
> ⚠️ Mobile first — always verify small screen layout
> ⚠️ No real client data or photos
> ⚠️ GSAP must be initialized inside useEffect — avoid SSR issues
> ⚠️ Always use ScrollTrigger.refresh() after page load
> ⚠️ Always use prisma migrate — never db push
