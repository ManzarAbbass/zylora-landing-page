# Zylora Agency Educational Videos Section Spec

## Overview
This is an interactive content streaming feature module for the Zylora agency landing page layout [cite: 1]. Use this specification document to structure the UI component architecture, horizontal media grids, and high-contrast thumbnail layouts for OpenCode execution [cite: 1]. This section bridges editorial marketing layouts with authentic social platform visual language [cite: 1].

## Component Requirements & Layout Architecture

### 1. Section Header Block (Asymmetric Grid)
- **Section Surface**: Clean Solid White background (`#FFFFFF`) [cite: 1].
- **Left Column Layout Focus**:
  - **Eyebrow Element**: Left-aligned layout pill badge containing exactly the descriptive string: `"EMAIL MARKETING INSIGHTS"` (Fixed inconsistency from the reference template to accurately map current section theme) [cite: 1].
  - **Primary Header (H2)**: Large editorial Serif font face running across a left-aligned layout [cite: 1]. Text must read exactly: `"Over 35 *Educational* Videos"`. The specific word *“Educational”* must use markdown italicization markers to toggle color configurations into the custom purple brand accent channel [cite: 1].
  - **Subtext Block**: A clean minimal description string positioned directly beneath the H2 reading exactly: `"Checkout the latest in the E-commerce Email Marketing space."` [cite: 1].
- **Right Column Layout Focus**:
  - **Action Element**: Top-right aligned solid purple CTA button. Styling properties must remain completely identical to Section 1 layout tokens to maintain design rhythm [cite: 1].

### 2. Three-Column Native Video Thumbnail Grid
A balanced 3-column media card container on desktop viewports, responsive-configured to collapse cleanly into a swipeable horizontal carousel track on mobile screen viewports [cite: 1].

#### Shared Media Card Node Stack
Each card module must expose the following layered component infrastructure from top to bottom [cite: 1]:
- **Primary Video Thumbnail Canvas**: A prominent thumbnail aspect ratio window styling that intentionally mimics high-converting YouTube platform visual styles [cite: 1]. Must support:
  - High-contrast, bold sans-serif text hooks overlaid onto the graphic [cite: 1].
  - Presenter reaction-face transparency cuts layer styling over background fills [cite: 1].
- **Persistent Promo Banner**: A solid colored banner/bar positioned directly beneath the thumbnail window block [cite: 1]. All 3 cards share an identical persistent text copy value acting as a global giveaway CTA overlay [cite: 1].
- **Attribution Node**: A clean, minimal channel/creator attribution text string aligned underneath the promotional banner element [cite: 1].
