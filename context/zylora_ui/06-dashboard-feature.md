# Section: Custom Software Dashboard Feature

## Layout
- Dark/black rounded section (contrasts with white/purple sections elsewhere — signals a distinct, "serious tech" moment on the page), faint repeating wordmark pattern in background like section 2.
- Pill badge top-left ("THE TRUTH ABOUT YOUR NUMBERS" — reused badge label from section 3, ties the two sections' argument together).
- Serif heading, 3 lines, 2 words highlighted in yellow italic ("Built", "Software"), referencing a competitor tool by name in the heading.
- Body paragraph below heading explaining the problem with the referenced tool and the custom solution built instead.
- Yellow CTA button top-right, same style as section 2's yellow CTA.
- Below: nested gray card containing a bolded sub-heading ("The Full Picture, One Screen") + short paragraph, then an embedded product screenshot (actual dashboard UI: sidebar nav + grid of metric stat-tiles with dollar figures and percentage deltas).
- Carousel dots below the screenshot (this is slide 1 of a multi-slide product-screenshot carousel).

## Content structure
- Argument structure: name the industry-standard tool → describe its limitation → introduce proprietary tool as the fix → show proof (screenshot of real-looking dashboard with specific revenue figures, campaign vs. flow split, deliverability, ROI).

## Design notes
- Color/tone shift to black+yellow (vs. purple used everywhere else) marks this as the page's "credibility peak" section — using a different palette to signal "pay extra attention here."
- Dashboard screenshot uses realistic-looking data density (multiple stat tiles, percentage deltas, date range selector) — this level of specificity is what makes a proprietary-tool claim feel credible.

## Reference-only note
This section's persuasive power comes entirely from the screenshot looking like a real, working product. If a section like this is ever built for a real project, the tool/dashboard shown must actually exist and the numbers must be real or explicitly marked as illustrative — a fabricated dashboard screenshot with specific dollar figures is the single most misleading pattern on this whole page.
